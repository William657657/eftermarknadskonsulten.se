2. Ändamål och rättslig grund

Vi behandlar uppgifter för att kunna leverera våra tjänster, hantera förfrågningar, uppfylla avtal och rättsliga skyldigheter. Rättslig grund är i första hand avtal, berättigat intresse och rättslig förpliktelse.

3. Kategorier av personuppgifter

Kontaktuppgifter, ärenderelaterat underlag, tidslinjer och kommunikation som krävs för att kunna ge rådgivning, framställa yttranden och föra ärenden framåt.

4. Mottagare

Vid behov delas uppgifter med samarbetspartner som jurister/verkstäder samt myndigheter som ARN. Endast nödvändiga uppgifter delas.

5. Lagringstid

Vi sparar uppgifter så länge som krävs för ändamålet och enligt tillämplig lagstiftning. Därefter raderas eller anonymiseras de.

6. Dina rättigheter

Du har rätt till tillgång, rättelse, radering, begränsning, dataportabilitet och att invända. Kontakta oss via info@emksab.se.

7. Säkerhet

Vi vidtar lämpliga tekniska och organisatoriska åtgärder för att skydda uppgifterna.

8. Kontaktformulär via e-post

Kontaktformulär via e-post

När du kontaktar oss via formuläret på eftermarknadskonsulten.se öppnas din egen e-postklient med ett färdigifyllt meddelande. Ingen information skickas eller lagras på vår webbserver innan du själv väljer att skicka e-posten. Mottagare är alltid info@emksab.se.
